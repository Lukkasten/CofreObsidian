[[3. RECURSOS/IA - INTELIGENCIA ARTIFICIAL/Prompts/Prompt Provocador — Escrita com Valor de R$10.000\|Prompt Provocador — Escrita com Valor de R$10.000]]



🟦 1. Estruture uma apresentação inteira

“Preciso criar uma apresentação sobre [tema].
O público é [tipo de público].

Crie a estrutura completa de uma apresentação persuasiva com 12–15 slides.
Inclua o título de cada slide e os pontos principais.”

------

🟦2. Transforme uma ideia em slides

“Transforme esta ideia em uma apresentação clara e persuasiva:
[explique sua ideia].

Crie slides curtos, com títulos fortes e poucos pontos por slide.”

----

🟦3. Crie uma apresentação que vende

“Crie uma apresentação de vendas para meu serviço de [serviço].

A apresentação deve:
• mostrar o problema
• apresentar uma nova forma de pensar
• mostrar a solução
• e levar para a oferta.”

-----
🟦4. Transforme um texto em apresentação

“Pegue este texto e transforme em slides de apresentação claros e objetivos:
[cole o texto].

Cada slide deve ter um título forte e no máximo 3 pontos.”

-----

🟦5. Crie uma apresentação para reunião com clientes

“Preciso apresentar [tema] para empresários.

Crie uma apresentação simples que explique o problema, mostre oportunidades e termine com próximos passos.”

----


🟦6. Crie slides que prendem atenção

“Crie uma apresentação sobre [tema] com slides que gerem curiosidade.

Use títulos curtos e ideias que façam o público querer ouvir o próximo slide.”

-----

🟦7. Transforme dados em narrativa

“Tenho esses dados:
[cole os dados].

Transforme isso em uma apresentação que conte uma história clara.”

-----

🟦8. Crie uma apresentação estilo TED

“Crie uma apresentação estilo TED Talk sobre [tema].

A estrutura deve gerar reflexão, surpresa e uma ideia forte no final.”

----

🟦9. Simplifique uma apresentação complexa

“Tenho esse assunto complexo: [tema].

Explique como se fosse uma apresentação para iniciantes, usando exemplos simples.”

-----

🟦10. Crie slides que fecham negócios

“Crie os 5 slides finais de uma apresentação de vendas que levem o cliente a tomar decisão.

Inclua:
• resumo do problema
• solução
• prova
• convite para avançar.”
